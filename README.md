# English to be

Next.js 14 SaaS scaffold for an English learning app with:

- App Router + TypeScript
- Tailwind CSS + shadcn-style UI primitives
- Supabase auth/data
- LemonSqueezy payments
- Vercel-ready structure

## Setup

1. Copy `.env.example` to `.env.local`.
2. Fill Supabase and LemonSqueezy environment variables.
3. Install dependencies with `npm install`.
4. Run `npm run dev`.

## Supabase

Run SQL from [supabase/schema.sql](/D:/app/_in_dev_codex/eng_tobe_saas/supabase/schema.sql) to create tables and RLS policies.
